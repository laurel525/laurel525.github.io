
# RGB Loop Project!

Laurel Barrett


## Assignment Description 

This project uses "while" loops to mimic the action of lighting up LEDs of different colors. The runTimes can be any variable and has to print three of the same color in a row every 10th time through the loop. The pattern has to remain regardless of the runTimes.
